# Изображения

Поместите изображения статей и обложку журнала в эту папку.

## Структура:

- `cover.png` или `cover.jpg` - обложка журнала
- `hero-1.jpg`, `hero-2.jpg` и т.д. - главные изображения статей
- Другие изображения для статей

## Использование:

В файле `app/data/articles.ts` укажите пути к изображениям:
- `/images/cover.png` - для обложки
- `/images/hero-1.jpg` - для изображений статей

Все пути должны начинаться с `/images/` и указывать на файлы в этой папке.
