# PWA Icons

Поместите иконки приложения в эту папку:

- `icon-192.png` - 192x192 пикселей
- `icon-512.png` - 512x512 пикселей

Иконки должны быть квадратными PNG-файлами.

## Рекомендации:

- Используйте простой, узнаваемый дизайн
- Иконка должна хорошо выглядеть на светлом и темном фоне
- Для maskable иконок оставьте безопасную зону (padding) вокруг основного контента

## Генерация иконок:

Можно использовать онлайн-инструменты:
- PWA Asset Generator
- RealFaviconGenerator
- Или создать вручную в графическом редакторе
