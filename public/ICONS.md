# PWA Icons

Для работы PWA необходимы иконки приложения.

Создайте следующие файлы в папке `public/`:
- `icon-192.png` (192x192 пикселей)
- `icon-512.png` (512x512 пикселей)

Иконки должны быть квадратными PNG-файлами с прозрачным или непрозрачным фоном.

Для быстрого создания можно использовать:
- Онлайн-генераторы (например, PWA Asset Generator)
- Графические редакторы (Figma, Photoshop, etc.)
- Простые инструменты командной строки
