# Как добавить изображения

## Шаги:

1. **Создайте папку для изображений** (если её еще нет):
   ```
   public/images/
   ```

2. **Поместите изображения в папку `public/images/`**:
   - Обложка журнала: `cover.png` или `cover.jpg`
   - Изображения статей: `hero-1.jpg`, `hero-2.jpg` и т.д.

3. **Обновите пути в данных**:

   **Обложка** - в файле `app/data/issue-config.ts`:
   ```typescript
   export const issueConfig = {
     coverImage: '/images/cover.png', // или '/images/cover.jpg'
     issueNumber: 'Летний выпуск 2025',
     title: 'МедиаБосс'
   };
   ```

   **Изображения статей** - в файле `app/data/articles.ts`:
   ```typescript
   {
     slug: "article-slug",
     title: "Название статьи",
     author: "Автор",
     heroImage: "/images/hero-1.jpg", // путь к изображению
     content: "..."
   }
   ```

## Пример структуры:

```
public/
  images/
    cover.png          ← Обложка журнала
    hero-1.jpg         ← Изображение для первой статьи
    hero-2.jpg         ← Изображение для второй статьи
    ...
```

## Важно:

- Все пути должны начинаться с `/images/`
- Поддерживаются форматы: `.jpg`, `.jpeg`, `.png`, `.webp`
- Если изображение не найдено, оно автоматически скроется
- Размеры изображений оптимизируются автоматически браузером

## После добавления изображений:

1. Перезапустите dev сервер: `pnpm dev`
2. Или пересоберите проект: `pnpm build`
